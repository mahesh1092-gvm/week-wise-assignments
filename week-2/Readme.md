### Modules/packages
-> modules are files which are created by others can be reused by us.
-> they are used for better code writing and for removing clumsyness of large code.
-> using modules build ecommerce application with app,cart,discount,payment,product 5 different files importing and exporting the necessary data.

-> operations on predfined methods:

  map()->select and modify

  find()->find and return value, if not found return undefined.
